#  AI Study Planner (Smart Python Project)

##  Project Description

This project is a smart study planner developed using Python. It helps students manage their study time effectively by analyzing their marks and automatically generating a personalized study schedule.

##  Features

* Add subjects and marks
* Automatically generate study timetable
* Identifies weak subjects
* Suggests where to focus more
* Simple and user-friendly interface

##  Working Logic

The system uses a weighted logic approach:

* Subjects with lower marks are given more study time
* Subjects with higher marks are given less time
* Total available study hours are distributed accordingly

##  Technologies Used

* Python
* Basic Data Structures (List, Dictionary)
* Logic Building
